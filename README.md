# Pixel OS #

### Sync ###

```bash

# Initialize local repository
repo init -u https://github.com/PixelOS-and-Not-So-Pixel/manifest -b eleven

# Sync
repo sync -c -j$(nproc --all) --force-sync --no-clone-bundle --no-tags
```

### Build ###

```bash

# Set up environment
$ . build/envsetup.sh

# Choose a target
$ lunch aosp_$device-userdebug

# Build the code
$ mka bacon -jX
```
